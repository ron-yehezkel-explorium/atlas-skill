# Team Timeline Capacity

Canonical local capacity and calendar rules for team timeline. Stored under `reference/team-timeline/`.

## Rules

- This file is local-only. Never overwrite it during Jira sync.
- Use whole days only.
- `0` day entries are invalid.
- Weekends and explicit non-working days block scheduling.
- Capacity events render as visible gantt bars.

## Calendar Settings

- timezone: Asia/Jerusalem
- working_days: Sunday, Monday, Tuesday, Wednesday, Thursday
- weekend_days: Friday, Saturday
